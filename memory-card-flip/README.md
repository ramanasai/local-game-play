# Memory Card Flip

A fast, responsive **Memory Card Flip** game with a **Golang backend** (SQLite) and a **React + TypeScript** frontend, using **Bun** for the JS toolchain.

## Tech Stack

*   **Frontend:** React + TypeScript + Vite, Tailwind CSS, Zustand, React Router.
*   **Runtime:** Bun 1.x.
*   **Backend:** Go 1.22+, Chi router, SQLite.

## quickstart

```bash
# Install frontend deps
make install

# Run dev server (FE + BE)
make dev
```
